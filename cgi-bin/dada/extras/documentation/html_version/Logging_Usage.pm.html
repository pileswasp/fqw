NAME

DADA::Log

DESCRIPTION

        my $log = new DADA::Logging::Usage;
        
        $log->mj_log($list, $action, $details); 
        $log->trace("something's happening");

SYNOPSIS

This simple module allows simple logging utilities to Dada Mail. It also allows you to 'trace' whatever you want really in Dada Mail, This is best for bug fixing and usually isn't good for general use.
